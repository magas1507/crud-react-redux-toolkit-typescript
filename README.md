# CRUD com React, Redux, TypeScript, Tailwind CSS e Tremor

Um aplicativo CRUD desenvolvido com **React**, **Redux Toolkit**, **TypeScript**, **Tailwind CSS**, **shadcn/ui** e **pnpm**. Implementa uma interface responsiva para gerenciar dados, com foco em performance e boas práticas.

## Funcionalidades

- Criação, leitura, atualização e exclusão de itens.
- .
- Gerenciamento de estado com Redux Toolkit.
- Tipagem forte com TypeScript.

## Tecnologias

- React 18
- Redux Toolkit
- TypeScript
- Tailwind CSS 4.x
- shadcn/ui
- pnpm 10.x
- Vite
